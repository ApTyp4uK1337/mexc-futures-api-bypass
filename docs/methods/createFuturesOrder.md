# createFuturesOrder

Create a new futures order.

- **Endpoint:** `/v1/createFuturesOrder`

---

## üì• Request parameters

| **Parameter (PHP)**        | **Sent as (API)**       | **Type**    | **Required** | **Description**                                                                 |
|----------------------------|--------------------------|-------------|--------------|---------------------------------------------------------------------------------|
| `symbol`                   | `symbol`                 | `string`    | ‚úÖ          | Name of the contract (e.g., `BTC_USDT`).                                       |
| `price`                    | `price`                  | `decimal`   | ‚úÖ          | Order price.                                                                   |
| `type`                     | `type`                   | `int`       | ‚úÖ          | Order type:<br>1 = Limit<br>2 = Post Only<br>3 = Fill or Kill<br>4 = All or None<br>5 = Market<br>6 = Market to Current |
| `open_type`                | `openType`               | `int`       | ‚úÖ          | Margin type: `1` = Isolated, `2` = Cross.                                      |
| `position_mode`            | `positionMode`           | `int`       | ‚ùå           | Position mode: `1` = Hedge, `2` = One-way. Defaults to user's config.          |
| `side`                     | `side`                   | `int`       | ‚úÖ          | Direction: `1` = Open Long, `2` = Close Short, `3` = Open Short, `4` = Close Long. |
| `vol`                      | `vol`                    | `decimal`   | ‚úÖ          | Order volume.                                                                  |
